scPred integration with Seurat

Jan 16, 2019 11 min read

In this tutorial, I’ll show how to incorporate prediction analyses using Seurat along with scPred.

First, we’ll load scPred and the tidyverse:

library(scPred)
library(tidyverse)

Let’s load Seurat:

library(Seurat)

We’ll work with a dataset obtained to analyze the ventral midbrain development in human (see Molecular Diversity of Midbrain Development in Mouse, Human, and Stem Cells). The data will be downloaded from Hemberg’s lab:

download.file("https://scrnaseq-public-datasets.s3.amazonaws.com/scater-objects/manno_human.rds", destfile = "~/Downloads/manno_human.rds")

Then we import the data and conver thescater object into Seurat

midbrain <- readRDS("~/Downloads/manno_human.rds")
midbrain <- Convert(from = midbrain, to = "seurat")

To show the training and prediction steps we’ll split the data into two partitions. First, we generate random indices to select 75% of the total number of cells for training:

n <- length(midbrain@cell.names)
set.seed(66)
i <- sample(seq_len(n), size = n * 0.75)

Using this indices, we can subset the Seurat object to create two objects containing the training and test data.

trainData <- SubsetData(midbrain, cells.use = midbrain@cell.names[i])
testData <- SubsetData(midbrain, cells.use = midbrain@cell.names[-i])

We normalize and scale the data using Seurat. The most variable genes are also identified in order to perform dimensionality reduction.

trainData %>% 
  NormalizeData(normalization.method = "LogNormalize", 
                scale.factor = 10000) %>% 
    FindVariableGenes(mean.function = ExpMean, 
                      dispersion.function = LogVMR, 
                      x.low.cutoff = 0.0125, 
                      x.high.cutoff = 3, 
                      y.cutoff = 0.5, 
                      do.plot = FALSE) %>% 
    ScaleData() -> trainData
## Scaling data matrix

Principal component analysis

scPred finds a set of discriminative principal components as features to perform the predictions. Using Seurat, we’ll perform a PCA and visualize the results.

trainData <- RunPCA(trainData, do.print = FALSE)

For this tutorial we’ll focus on predicting the source of the dopaminergic neurons (ESCs - embryonic stem cells, iPSCs - induced pluripotent stem cells, and ventral midbrain cells)

PCAPlot(trainData, group = "Source")

Get feature space

The function getFeatureSpace() from scPred can handle Seurat objects as input. The Seurat object must contain a principal component analysis computed already. Here, we provide the trainData object and specify the variable in the metadata slot containing the cell class of interest (e.g. Source).

scpred <- getFeatureSpace(trainData, pVar = "Source")
## Not all the classes are valid R variable names
## The following classes are renamed: 
## ventral midbrain
## 
## See new classes in 'Source.valid' column in metadata:
## ventral.midbrain
## 
## DONE!

Once we obtain the feature space, the output of the getFeatureSpace() function is a scPred object which contains the normalized expression data, the principal component analysis and the metadata.

Model training

Now we’ll train a support vector machine with a radial kernel using scPred using the trainData function.

scpred <- trainModel(scpred)

Prediction

The scPredict() function can also handle Seurat objects containing the gene expression data of the cells to be predicted. This functions returns a dataframe stored in the @predictions slot with the class probabilities and the predicted class for each cell in the test dataset.

scpred <- scPredict(scpred, newData = testData)
## Scaling data matrix
## Predicting cell types

In order to get the accuracy of scPred, well add the test metadata to the @predMeta slot. The number of rows in the metadata must match the number of cells that were predicted.

scpred@predMeta <- testData@meta.data

Once we have the test metadata stored in the scPred object, we can use the crossTab function to get a contingency table showing the predictions versus the true cell classes. In this case, the column Source of the test metadata contains the true cell labels:

We reorder the rows of the contingency table to match the true cell categories

crossTab(scpred, "Source")[c(1,2,4,3),]

## iPSCs            0.00  0.97                0
## ventral.midbrain 0.00  0.00                1
## unassigned       0.03  0.00                0
